Projet ARTISHOW

Modèles d’IA générative pour la création d’images simples

Présentation

Ce projet vise à explorer l'utilisation de modèles d'IA générative pour générer automatiquement des images simples à partir de texte ou de manière non guidée. Une interface web interactive permet d'expérimenter différents modes de génération, accompagnés de contenus pédagogiques pour mieux comprendre les modèles utilisés.

Génération par prompt

Sur cette page, vous pouvez entrer une description textuelle pour générer une image contenant des formes géométriques simples : cercles, carrés, triangles, losanges, disques. Vous pouvez contrôler la taille, la couleur et la position.

Exemples de prompts :

  • un carré vert en bas à droite
  • un cercle rouge en haut à gauche et un triangle bleu au centre
